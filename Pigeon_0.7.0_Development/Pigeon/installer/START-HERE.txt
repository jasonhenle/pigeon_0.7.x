Pigeon @VERSION@ — Raspberry Pi
================================

There is only ONE installer to use:

    Install-Pigeon

Steps:
  1. Open the @APP_DIR@/installer folder (the folder created when you extract the tarball).
  2. Double-click Install-Pigeon.
  3. Choose Execute (or Execute in terminal).
  4. Click Continue in the dialog.
  5. A black window opens — wait 5–15 minutes. Do not close it.
  6. When it says "Install finished successfully", press Enter.

Then use Run-Pigeon on your Desktop to launch Pigeon.

Metadata and TMDb artwork (IMPORTANT on Pi)
-------------------------------------------
The Pi does not get your Mac's TMDb key or Apple TV pairing automatically.

Easiest fix: copy your Mac state folder to the Pi:
  scp -r ~/.pigeon_0_6 pi@YOUR_PI_ADDRESS:~/.pigeon_0_6

Or read installer/setup/README.txt before installing.

After running Pigeon once, check logs on the Pi:
  ~/.pigeon_0_6/pigeon.log

If install fails with "E: failed to fetch":
  - The Pi is not online or cannot reach its software mirrors.
  - Connect Wi‑Fi (top-right) or plug in Ethernet.
  - Open the web browser on the Pi and confirm a website loads.
  - Menu → Preferences → Raspberry Pi Configuration → set correct date/time.
  - Open Terminal and run:  sudo apt update
  - When that succeeds with no errors, run Install-Pigeon again.

Ignore these other files (not for Pi):
  install_pigeon.command   (Mac only)
  Install-Pigeon.desktop   (optional; use Install-Pigeon instead)
