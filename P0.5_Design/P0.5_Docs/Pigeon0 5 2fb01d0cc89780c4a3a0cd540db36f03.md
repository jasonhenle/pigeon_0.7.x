# Pigeon0.5

Pigeon is an app and optionally, also a device. The function of Pigeon is to provide a home theater owner with fun insights and widgets to accompany their home theater. 

- Pigeon will ultimately have three modes:
    1. iPadOS local mode: This mode displays info based on content that’s currently playing on your iPad
    2. iPadOS network version: This uses your iPad to display realtime information from a network-enabled home theater
    3. iPadOS remote version: This version uses an iPad to control a Raspberry Pi that’s running Pigeon (this version will be coming later)

- General instruction for launching Pigeon
    1. Using [P0.5_detectDisplay](Pigeon0%205/P0%205_detectDisplay%202fb01d0cc89780b1ad0feb92ce09d2e9.md)
        1. Detect a currently active display, if there are more than one, use the primary one
        2. Find the resolution for the active display
        3. Store the resolution for the active display in [P0.5_userSettings](Pigeon0%205/P0%205_userSettings%202fb01d0cc89780238a22ee684f73be10.md) 
    2. Pigeon’s orientation can be determined by the active display’s resolution, but it also can be determined by a manual user selection
        1. Check to see if [P0.5_userSettings](Pigeon0%205/P0%205_userSettings%202fb01d0cc89780238a22ee684f73be10.md) contains any previous selections for orientation
            1. If [P0.5_userSettings](Pigeon0%205/P0%205_userSettings%202fb01d0cc89780238a22ee684f73be10.md) shows that the current display has been used before, check to see if there are any manually selected orientation options.
                1. If manual orientation selections are found use them
                2. If no manual orientation selections are found, determine orientation based upon the rules listed below.
                    1. If x-resolution > y-resolution; use [P0.5_landscape](Pigeon0%205/P0%205_userSettings/P0%205_landscape%202fb01d0cc89780c0af88e4a23bdf6100.md) 
                    2. If x-resolution == y-resolution; use [P0.5_landscape](Pigeon0%205/P0%205_userSettings/P0%205_landscape%202fb01d0cc89780c0af88e4a23bdf6100.md) 
                    3. If x-resolution < y-resolution; use [P0.5_portrait ](Pigeon0%205/P0%205_userSettings/P0%205_portrait%202fb01d0cc89780a4910bc66ebfc15033.md) 
    3. Crop [P0.5_pigeonCanvasCrop](Pigeon0%205/P0%205_pigeonCanvasCrop%202fb01d0cc897809f90afcebd408f10c3.md) to match the aspect ratio of the display
    4. For a test, use P0.5_GridTest.png to put put on [P0.5_pigeonCanvas](Pigeon0%205/P0%205_pigeonCanvas%202fb01d0cc89780ea95f5ee39bdc4e59f.md) 
        1. iCloud Drive / Desktop / Pigeon0.5 / Pigeon0.5_Assets / Test / P0.5_GridTest.png
    5. If a resolution change is detected, update [P0.5_pigeonCanvas](Pigeon0%205/P0%205_pigeonCanvas%202fb01d0cc89780ea95f5ee39bdc4e59f.md) to reflect.
